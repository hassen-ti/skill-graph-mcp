🔧 MCP Tools

search_skills(query)
Semantic search over 732 skills using OpenAI embeddings + Neo4j vector index. Returns top-3 candidates with semantic & hub scores.
vector search
get_skill(id, depth)
Retrieve the full expert context for a skill including instructions, tools, and graph neighbors. shallow = L1, full = L1+L2.
rate-limited 10/session
navigate(from_id, edge_type)
Traverse typed edges from any node. Supports REQUIRES, ENABLES, COLLABORATES_WITH, USES, EXTENDS, PART_OF.
graph traversal
get_knowledge(ref)
Safe file read from the knowledge base. 3-layer path confinement with extension whitelist and symlink resolution.
path-confined

🗂 Graph Schema

(:Skill) node properties

  • id — unique, lowercase, hyphens
  • name — human-readable label
  • description — plain text
  • type — domain / tool / role / cluster
  • hub_score — degree centrality [0–1]
  • embedding — Float[1536] OpenAI vector
  • payload_json — instructions + tools

[:EDGE_TYPE] relationships

  • REQUIRES — A cannot function without B
  • ENABLES — A unlocks or enhances B
  • COLLABORATES_WITH — semantic similarity
  • USES — A leverages B as a tool
  • EXTENDS — A specialises B
  • PART_OF — A belongs to cluster B

🚀 Quick Start

1

Clone & install

git clone https://github.com/Hassen-Ti/skill-graph-mcp.git
cd skill-graph-mcp
pip install -e .
2

Configure environment

cp .env.example .env
# Fill in OPENAI_API_KEY and NEO4J_PASSWORD
3

Start Neo4j via Docker

docker-compose up -d
# Neo4j UI → http://localhost:7474  (wait ~30s)
4

Connect to Claude Desktop

Add to %APPDATA%\Claude\claude_desktop_config.json:

{
  "mcpServers": {
    "skill-graph": {
      "command": "/path/to/python",
      "args": ["-m", "server.main"],
      "cwd": "/path/to/skill-graph-mcp",
      "env": {
        "NEO4J_URI": "bolt://localhost:7687",
        "NEO4J_USER": "neo4j",
        "NEO4J_PASSWORD": "your_password",
        "OPENAI_API_KEY": "sk-..."
      }
    }
  }
}

Restart Claude Desktop — the MCP appears alongside your other servers.
